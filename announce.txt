The following package has been uploaded to the Cygwin distribution:

* biber-2.5-1

Biber is a BibTeX replacement for users of BibLaTeX.  Biber supports full UTF-8, can (re-)encode input and output, supports highly configurable sorting, dynamic bibliography sets, and many other features.

This is an update to the latest upstream release.  See /usr/share/doc/biber/CHANGES for a list of changes.
